# Sound Effects Directory

Place sound effect files here for pack opening animations.

## Required Files

- `cash-register.mp3` or `cash-register.wav` - Cash register sound effect for Rare/Mythic card pulls

## Recommended Sources

- [Pixabay](https://pixabay.com/sound-effects/search/cash-register/)
- [Freesound](https://freesound.org/search/?q=cash+register)
- [Zapsplat](https://www.zapsplat.com/sound-effect-categories/cash-register-sounds/)

## File Format

- Format: MP3 or WAV
- Duration: 1-3 seconds recommended
- Volume: Normalized (will be adjusted in code)
